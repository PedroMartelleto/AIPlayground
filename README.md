## Getting Started

### Prerequisites

> [Node](https://nodejs.org)

### Installing

Clone the repository

```
git clone https://github.com/pedromartelleto/aiplayground.git
```

Install dependencies

```
npm install
```

Run

```
npm start
```

## Running the tests

```
npm test
```
